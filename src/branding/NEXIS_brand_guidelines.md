
# Nexis Brand Guidelines

## Logo Concept
Nexis represents a dynamic overlay engine connecting multiple live platforms.
The open frame symbolizes an overlay container, while internal modules represent
dynamic widgets such as counters, progress bars, and notifications.

## Core Colors
Background: #0B0F14
Primary Accent: #2FB7FF
Secondary Accent: #A64DFF
Alert / Counter Accent: #FF8C3A
Interface Frame: #E6E8EA

## Typography
Recommended UI Fonts:
- Inter
- Source Sans 3
- IBM Plex Sans

Logo wordmark style:
Geometric sans‑serif with wide tracking.

## Usage Rules
Maintain sufficient contrast against dark backgrounds.
Prefer the full‑color version for marketing material.
Use monochrome variants for UI contexts requiring minimal visual noise.

## Icon Sizes
Provided sizes:
16, 32, 48, 64, 128, 256, 512, 1024 px

Use:
- 16–32 px → favicon / browser tabs
- 64–128 px → UI icon
- 256–512 px → application icon
- 1024 px → marketing / store assets

## Clear Space
Minimum margin around the icon: 20% of logo width.
